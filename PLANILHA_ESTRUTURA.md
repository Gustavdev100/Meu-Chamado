# 📊 Estrutura da Planilha Google Sheets

## ✅ Criar Nova Planilha

1. Acesse: https://sheets.google.com
2. Clique em "Criar" → "Planilha em branco"
3. Nome: **Meu Chamado**
4. Clique em "Criar"

---

## 📋 Estrutura (Copie os Headers Exatamente)

Na **linha 1**, copie essas 21 colunas em ordem:

```
ID | Tipo | Status | Título | Descrição | Cidade | Base | Nome Solicitante | E-mail Contato | Prioridade | Itens | MID Localização | MID Tipo Material | Categoria Compra | Observações Admin | Foto URL | Prazo Visita | Prazo Orçamento | Prazo Entrega | Prazo Busca | Data Criação
```

### Instrução Prática:
1. Clique na célula **A1**
2. Cole exatamente isso:
```
ID	Tipo	Status	Título	Descrição	Cidade	Base	Nome Solicitante	E-mail Contato	Prioridade	Itens	MID Localização	MID Tipo Material	Categoria Compra	Observações Admin	Foto URL	Prazo Visita	Prazo Orçamento	Prazo Entrega	Prazo Busca	Data Criação
```
3. Pronto!

---

## 🔑 Informações da Planilha

Depois que criar, **COPIE o ID da URL**:

```
https://docs.google.com/spreadsheets/d/AQUI_ESTA_O_ID/edit
```

Você precisará desse ID para o script.
